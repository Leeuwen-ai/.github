# Leeuwen .github — Agent Conventions

## Project

Leeuwen (named after Antonie van Leeuwenhoek) builds **financial world models**: generative simulators that learn the underlying structure of markets from multimodal data.

## Scope

This repo contains the **GitHub org profile** — the public landing page for the Leeuwen-ai organization. Logo assets and the profile README only.

## Conventions

- Keep the profile README minimal and up to date
- Logo SVGs in `profile/` — maintain both light and dark theme variants
- This repo is **public** — never add proprietary information, internal links, or private repo references

## Git Workflow

- Always `git pull` before starting work to avoid conflicts
- Always `git push` promptly after committing to keep the remote up to date

## Security

- This is a **public** repo. Never commit anything sensitive here.
- No internal URLs, team details beyond what's already public, or references to private repos.
