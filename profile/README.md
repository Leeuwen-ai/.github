**Financial world models.** Generative simulators that learn the underlying structure of markets from multimodal data — a foundation for training, evaluating, and deploying trading agents.

Named after [Antonie van Leeuwenhoek](https://en.wikipedia.org/wiki/Antonie_van_Leeuwenhoek), who built his own microscopes to see what others couldn't.
